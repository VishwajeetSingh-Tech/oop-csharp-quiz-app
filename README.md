# 🧩 Interactive Quiz Application — C# WinForms

## 📌 Overview
A Windows Forms desktop quiz application built in C# featuring 29 multiple-choice questions, real-time score tracking, and dynamic question navigation. Demonstrates event-driven programming, UI design, and data structures in C#.

## 🏥 Healthcare IT Relevance
- Event-driven UI design applicable to clinical desktop applications
- Data structures (2D arrays) used in healthcare assessment tools
- Score tracking logic mirrors patient assessment scoring systems
- Windows Forms UI patterns used in hospital management software

## 🎮 Features
- 29 multiple choice questions across various topics
- 4 answer options per question (A, B, C, D)
- Real-time question counter (e.g. Question 3/29)
- Answer validation with error handling
- Final score display on completion
- Quiz reset functionality for multiple attempts

## 💡 Programming Concepts Demonstrated
- 2D Arrays — Quiz data stored as string matrix
- Event Handling — Button click events for Start and Next
- Conditionals — Answer checking logic
- UI Controls — Labels, RadioButtons, Buttons
- String Formatting — Dynamic question counter display
- Error Handling — MessageBox validation for unanswered questions

## 📁 Project Structure
- Form1.cs — Main quiz logic and event handlers
- Form1_Designer.cs — UI layout and control definitions
- Program.cs — Application entry point

## 💻 Technologies Used
- C# .NET 8
